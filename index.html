Enhanced Intrusion Detection in Network Traffic Using Deep Learning

Present Project: This ongoing final year project focuses on using deep learning to enhance Intrusion Detection Systems (IDS) for detecting network traffic anomalies. The system uses Long Short-Term Memory (LSTM) networks to detect both known and unknown network intrusions in real-time. By modeling the sequential nature of network traffic, it identifies patterns that may indicate malicious behavior. The system is trained on publicly available datasets, including CICIDS-2023, to classify network traffic as normal or malicious and detect various types of attacks. The project is currently in progress with ongoing model optimization and real-time implementation.
